The learning path

From first blocks to real Python

The program grows with the student — each stage builds naturally on the last, so coding always feels achievable and never overwhelming.

Block coding

Drag-and-drop programming on Scratch and Micro:bit builds core logic — sequences, loops, and conditions — with instant, visual results.

Text-based coding

The leap into real code with Python — variables, functions, and logic that power genuine programs.

Apps & games

Students apply their skills to build playable games, simple apps, and interactive projects they're proud to share.

Python & AI

Advanced students use Python to explore data, automation, and the basics of artificial intelligence.

Grade-wise progression

The right level for every learner

Coding is taught in three stages so a Primary student and a Senior student each get work that's exciting and achievable for them.

Primary (Grades 1–5)

Block coding & logic

Colourful, visual programming on Scratch and Micro:bit. Students build animations and simple games while learning the core ideas of sequencing and logic.

Intermediate (Grades 6–8)

Into real code

The bridge from blocks to text-based programming — students start writing Python and building genuine apps and games of their own.

Senior (Grades 9–10)

Python, data & AI

Real-world programming — automation, working with data, and an introduction to AI and machine-learning concepts through hands-on projects.
